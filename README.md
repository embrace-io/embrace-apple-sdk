# Embrace Apple SDK

[![Apache-2.0](https://img.shields.io/badge/license-Apache--2.0-orange)](./LICENSE.txt)

The Embrace Apple SDK builds on top of [OpenTelemetry](https://opentelemetry.io) to capture performance data for
iOS/iPadOs/tvOS/visionOS/watchOS apps and provides user-focused observability of your system by connecting mobile and backend telemetry seamlessly.

# Getting Started

Please follow our [SDK integration instructions](https://embrace.io/docs/ios/6x/) to get started with Embrace in your app.
If you don't have one already, you can [create an account here](https://dash.embrace.io/signup/).
